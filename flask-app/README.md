Python / Flask app
==================

### Technical Stack

	- Python: python 3.7

  - Flask: 1.1.1

	- Database: MongoDB 4.0.13

### Create virtual environment for Python 3.7
  ```
    $ python3.7 -m venv env
    $ source env/bin/activate
    $ pip install Flask
  ```

  or 

  ```
    $ pip install -r requirements.txt
  ```

### Run server
  ```
    $ python hello.py
  ```

### Server urls

  - http://localhost:5000
